# minetest-mod-invman
Allows to check and grab, remove items from players' inventories  
[![ContentDB](https://content.minetest.net/packages/Zemtzov7/invman/shields/downloads/)](https://content.minetest.net/packages/Zemtzov7/invman/)
### Usage
* `/invman <playername>` - open InvMan formspec targeted on online player
* There are three modes of managing items:
	* Copy - copy the item to your inventory
	* Take - move the item to your inventory
	* Remove - just remove item from target's inventory
* Use tabs to navigate through player's inventory lists
